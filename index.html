What's the story?

On November 20, AOL–the parent company of Nullsoft–posted the following notice on the Winamp website:

Winamp.com and associated Web services will no longer be available past December 20, 2013. Additionally, Winamp Media players will no longer be available for download. Please download the latest version before that date. See release notes for latest improvements to this last release. Thanks for supporting the Winamp community for over 15 years.

As a result, we have decided to copy all archived threads to preserve some of the valuable information stored there. In the process, the pages have been styled using Bootstrap 3 to ensure the best compatibility on a wide range of devices.

If you prefer keeping a copy on your local hard drive, you can clone the repository... 

git clone https://github.com/NSIS-Dev/NSIS-Forums.git
...or download a zipped copy.
