Well, try the following:

StrCpy $1 33
StrCpy $2 88
; 33/88=0.375
IntOp $1 $1 * 1000
IntOp $3 $1 / $2 ; $3 is now 375

; Now try to output $3:
IntOp $4 $3 / 1000
IntOp $5 $3 % 1000
MessageBox MB_OK "$4.$5" ; output the number, in this case 0.375

Exchange 1000 with a greater value if you need more digits after the .
