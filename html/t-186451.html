Shouldn't it be like this?:

!defineMUI_PAGE_CUSTOMFUNCTION_PRESetupDefault_LanguageStrings

!defineMUI_PAGE_CUSTOMFUNCTION_SHOWSetupDefault

!defineMUI_PAGE_CUSTOMFUNCTION_LEAVESetupDefault_GetState

!insertmacroMUI_PAGE_INSTFILES

!insertmacroMUI_PAGE_FINISH
If your pre function is just going to set up your strings then I would include that in your show function anyway. Then I generally write the function like this:

PagecustomCustomPageSomethingLeaveCustomPageSomething

!insertmacroMUI_PAGE_INSTFILES

!insertmacroMUI_PAGE_FINISH
Which I think is the old way.
