in you function page put an INI head, something like this:


[Settings] 
NumFields=1

[Field 1]
Type=GroupBox 
Text=Tareas adicionales 
Left=0 
Top=0
Right=300  
Bottom=130

[PASS]
Pass=hereyourpasword

Then call it at runtime, something like this:

ReadINIStr $0 "$PLUGINSDIR\io.ini" "PASS" "Pass" ;the pass you set
