Section "language"

Subsection "German"
Setoutpath "$PROGRAMFILES"
file "C:\german.lang"
SubsectionEnd

Subsection "English"
Setoutpath "$PROGRAMFILES"
file "C:\english.lang"
SubsectionEnd

SectionEnd


this should work.
