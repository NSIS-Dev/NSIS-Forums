Might no work under 98/ME

OutFile 'test.exe'
ShowInstDetails show
AllowRootDirInstall true

page directory
page instfiles

Function .onInit
   ReadEnvStr $INSTDIR SYSTEMDRIVE
FunctionEnd

Section "boo"
   SetOutPath "$INSTDIR"
SectionEnd


Works in all Windows versions

!define MY_APP "My application Name"
OutFile 'test.exe'
ShowInstDetails show
AllowRootDirInstall true

page directory
page instfiles

Function .onInit
   StrCpy "$INSTDIR" "$WINDIR" 2
   StrCpy "$INSTDIR" "$INSTDIR\${MY_APP}"
FunctionEnd

Section "boo"
   SetOutPath "$INSTDIR"
SectionEnd
