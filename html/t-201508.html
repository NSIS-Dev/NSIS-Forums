SectionSetSize can dynamically set a section size. Something like the following should do:

Section "install this" id
#...
SectionEnd

Function .onInit
IfFileExists $EXEDIR\blah.exe 0 +3
SectionSetSize ${id} 500
Goto +2
SectionSetSize ${id} 2000
