[b]Problem solved[/b]
A "\" at the end of line tells makensis to concatenate the next line to the line with the "\" at the end.

Do this instead:


    Function temp ;Convert "\\" -> "\"
    Exch $R0
    DetailPrint "$R0"
    Pop $R0
    FunctionEnd

Function StrIOToNSIS
Exch $R0
Push $R1
Push $R2
Push $R3
Push $R4
StrLen $R2 $R0
${For} $R1 0 $R2
StrCpy $R3 $R0 2 $R1
${If} $R3 == "\r"
StrCpy $R4 "$\r"
${ElseIf} $R3 == "\n"
StrCpy $R4 "$\n"
${ElseIf} $R3 == "\t"
StrCpy $R4 "$\t"
${Else}
StrCpy $R4 ""
${EndIf}
${If} $R4 != ""
StrCpy $R3 $R0 $R1
IntOp $R1 $R1 + 2
StrCpy $R0 $R0 "" $R1
StrCpy $R0 "$R3$R4$R0"
IntOp $R2 $R2 - 1
IntOp $R1 $R1 - 2
${EndIf}
${Next}
Pop $R4
Pop $R3
Pop $R2
Pop $R1
Exch $R0
FunctionEnd

Function StrNSISToIO
