Use the .onVerifyInstDir callback...

Function .onVerifyInstDir

  StrCpy $0 $INSTDIR 2
  StrCmp $0 "C:" PathBad
  StrCmp $0 "D:" 0 PathGood
    PathBad:
    Abort
  PathGood:

FunctionEnd


Edit:
Or do you mean just the root dir?

In that case use this.
Function .onVerifyInstDir

  StrCmp $INSTDIR "C:" PathBad
  StrCmp $INSTDIR "D:" 0 PathGood
    PathBad:
    Abort
  PathGood:

FunctionEnd
