glory_man
27th January 2006 15:02 UTC

Use code like this:


InstallDir "c:\sec1\"
...
Function .onSelChange

...radiobutton code here...

!insertmacro SectionFlagIsSet ${sec1} ${SF_SELECTED} 0 sec2
StrCpy $INSTDIR "c:\sec1\"
goto end
sec2:
StrCpy $INSTDIR "c:\sec2\"
end:
FunctionEnd
