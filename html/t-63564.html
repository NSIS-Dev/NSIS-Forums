I forgot to tell the usage.. Here it is from the documentation attached to the installer...

What to add to the script :-
BrandBitmap "filename.bmp"
*********** Sets the branding bitmap.* This is necessary for this build else the default bitmap I used will be present... It MUST be a 16 color bitmap. The size MUST exactly be 90 x 245.

ProgressBarColors [Progress Bar Foreground Color]* [Progress Bar Background Color]

** *
**Specifies whether to use the default progress bar colors.* If you want to add different colors include this line. You have to include both the Progress bar foreground color and Progress bar background color.*

It is recommended to add the Gradient and also the Install colors to match
the bitmap and progress bar colors to look really good.
