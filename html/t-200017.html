StrCpy $R0 "1/15/2004"

Push $R0
Push "5"
Push "ab"
Call StrReplace
Pop $R0 ; $R0 = "1/1ab/2004"

You can grab the StrReplace Function from:
http://nsis.sourceforge.net/archive/...b.php?page=124
