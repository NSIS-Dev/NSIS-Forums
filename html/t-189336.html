StrCmp $1 "Update" UpdateFound
Goto UpdateFound

Is the same as:
StrCmp $1 "Update" UpdateFound UpdateFound
So it won't work.

You should put the command Goto right before the label UpdateFound:
 to go to the end. Like this below:

StrCmp $1 "Update" UpdateFound

  # UpdateNotFound codes
  
  Goto UpdateEnd
  
UpdateFound:

# UpdateFound codes

UpdateEnd:
