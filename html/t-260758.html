Heres an example of the command line I use to build an installer via a batch file

makensis /DBUILDFLAG=%C% /DVERSION=%V% /DREVISION=%R% "InstallerScript.nsi"

Within the script the /D values are referenced with:

${BUILDFLAG}
${VERSION}
${REVISION}
