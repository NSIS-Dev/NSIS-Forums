Defines are being processed on compile time. For example, when you add

!define TEMPVAR "$R0"

${TEMPVAR} will be replaced by $R0 when compiling.

If you want to use variables temporary, use Push/Pop:

Push $R0

...using $R0 as temp var...

Pop $R0
