There is no all in one ready to use solution. No install product is the same therefore you will have to do it yourself. It should be straight forward. Uninstall first (call the existing uninstaller) but leave configuration files or registry behind (uninstaller can have a command line switch for this). Then install but without overwriting configuration files or registry (a variable can be used).
