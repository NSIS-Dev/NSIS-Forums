ClearErrors

FileOpen$0 file.map

loop:

   FileRead $0 $1

    StrCmp$1 "" done

    StrCpy$2 $1 -4

    StrCmp$2 ".wal" 0 loop

        DetailPrint$1

        IfFileExists"$INSTDIR\\$1" loop

            DetailPrint "Warning: file doesn't exist!"

            Goto loop


error:

   Abort "Error"


>done:
