RestartRequired and RestartNotRequired are label identifiers which one must put in a script like so:

RestartRequired:
...
Goto Done
RestartNotRequired:
...
Done:

I would recommend using LogicLib.nsh though:
${If} $R0 == 1
...
${Else}
...
${EndIf}
