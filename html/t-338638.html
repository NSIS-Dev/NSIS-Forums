Download Latest Utility
I just finished the preliminary release of my DownloadLatest utility.

Sometimes you need to compile a package from a web source location that is static and file often changes (Adobe Flash anyone?) This utility when executed using the !system command will download the file if it needs to be and creates a handy NSH that includes the file's last modified date, ETag (used for proxy cache identification), file name, downloaded path, & version.

I included an example script that downloads the latest version of Adobe Flash x86 and generates a wrapper package for it.

This utility has proven invaluable in a corporate setting where I need to dynamically compile packages quickly when a change occurs.

See the Wiki Article at:
