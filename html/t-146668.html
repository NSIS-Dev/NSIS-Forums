Depends on the application. For NSIS:

Normal installers:

0 - Normal execution (no error)
1 - Installation aborted by user
2 - Installation aborted by script

Silent installers:

0 - Normal execution (no error)
1 - Installation aborted by user
1 - Installation aborted by script
