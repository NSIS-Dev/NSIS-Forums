Yes, set it's name to an empty string ("") using SectionSetText. For example:

Section"OnlyonXP"XPSection

#....

SectionEnd


Function.onInit

#getOSinto$0

StrCmp$0"XP"+3

SectionSetText${XPSection}""

SectionSetFlags${XPSection}0#unselectsection

FunctionEnd
