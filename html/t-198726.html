Ok, I found your problem.

You should be searching for the window title not window class...

FindWindow $0 "" "Temp"
 IntCmp $0 0 done
 IsWindow $0 0 done
 System::Call 'user32::PostMessageA(i,i,i,i) i($0,${WM_CLOSE},0,0)'
