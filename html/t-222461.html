Something like this:

        SetDetailsPrint none
        exec V:\someCmd.cmd
        SetDetailsPrint both
        DetailPrint "Running This command"

Use with last wiki ExecDos build


ExecDos::exec /NOUNLOAD /ASYNC "path\your_app.exe" "" "" ; or v:\somecmd.cmd if you want :)
DetailPrint "Running This command"
ExecDos::wait
Pop $0 ; exit code
