glory_man
9th June 2005 09:56 UTC

Example for filename.ext:


Section "SysFiles"
SetOutPath $INSTDIR
IfFileExists "$SYSDIR\filename.ext" +2 0 ; check if file present in sysfolder
File "filename.ext" ; install file in $INSTDIR if absent
SectionEnd
