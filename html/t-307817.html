I use this in my code
---
File "C:\MyWallPaper.bmp"
WriteRegStr HKCU "Control Panel\Desktop" "WallpaperStyle" "2"
System::Call 'user32::SystemParametersInfo(i 20,i 0,t "$INSTDIR\Graphics\MyWallPaper.bmp",i 3) i.r0'
---
