Search the forum first. You will find example like this.


Function .onInit
System::Call 'kernel32::CreateMutexA(i 0, i 0, t "App1") i .r1 ?e'
Pop $R0
StrCmp $R0 0 good
   MessageBox MB_OK "Application is already runing"
   Abort
good:
FunctionEnd
