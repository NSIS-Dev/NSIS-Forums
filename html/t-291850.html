NSIS Language Strings Translator
What is this?

It's a tiny tool that I've made to help me automate the translation of language strings.

How it works?

First load a file (nsi nsh txt) which contains language strings.
Then select a language from the drop down menu in which you want to translate the strings. The first discovered string appears in the upper text field. Enter the translation in the lower edit field and hit translate. Now the second discovered string appears, translate it, hit translate and so on until all strings are translated.
Now click save and you'll get a new NSIS header (nsh) with the translated strings, ready to include it in your script.

If anyone is interested to give a try and report bugs or suggest features, here it comes :)
