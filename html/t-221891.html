New plugin Skinned Button 1.0 released
Hello,

I am the developper of UltraModernUI, a new user interface with a style like the most recent installers for NSIS, I have just release a new plugin for NSIS named skinned button which allows to skin all the buttons of your installer. It's based on the wansis plgin of saivert.
It was developed with an aim of integrate it in the next version of UltraModernUI (version 1.0 beta 2)

1/ the Desktop always disappear for a good second, and then painfully redraw itself (uh?)
