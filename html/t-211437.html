VPatch 3 Release Candidate 6
VPatch 3 has been updated to Release Candidate 6, and I think it really deserves the RC label this time.

You can download it from: http://www.tibed.net/files/vpatch3releasecandidate.zip

Kichik, can the new updated NSIS plug-in be included into CVS?
It can be found in the archive under "VPatch\Source\NSISPlugin" and all the paths are 'set' for the NSIS Contrib path location and building into the plug-ins folder.
The plug-in is completely backwards compatible, but newly generated patches in v3 need this updated plug-in to leverage the new features such as MD5 checksums and file date preservation. Now that 2.06 is out this is a good time to put it in CVS.
