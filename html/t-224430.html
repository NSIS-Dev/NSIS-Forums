Yeah, before you load the ini file for a custom page (with InstallOptions::initDialog), just do a WriteINIStr to the field you want.


Function CustomPre
  WriteINIStr "myfile.ini" "Field 1" "Text" "New Text"

  InstallOptions::initDialog
  Pop $R0
FunctionEnd

With an INI file of

[Settings]
NumFields=1
[Field 1]
Type=Label
Text=bla
...
