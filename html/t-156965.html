Matrix(a,b) (c=-1; x={}; d=#[l(a)>l(b), l(a), l(b)]; #{c++<d, x[c]=a[c]+b[c]}; x)

tips:
1. no = needed between Matrix() and ().
2. x array should be declared before use.
3. if you'd like x to be the function output, place it as last expression.
