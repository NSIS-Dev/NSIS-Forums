If using the Modern UI:

!defineMUI_CUSTOMFUNCTION_GUIINITBlackBranding


FunctionBlackBranding

GetDlgItem$0$HWNDPARENT1028

EnableWindow$01

FunctionEnd




                
                
                

        
If using ExperienceUI:

        PHP Code:
        
                

                
                
                        
!defineXPUI_BRANDINGTEXT"NullsoftInstallSystem${NSIS_VERSION}"
(Actually this will use branding text colors defined in current skin - highly recommended)
See ExperienceUI docs for how to change Branding Text color

If using classic UI:
PHP Code:
Function.onGUIInit
GetDlgItem$0$HWNDPARENT1028
EnableWindow$01
FunctionEnd
