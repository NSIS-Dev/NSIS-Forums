StrCpy $0 3da06e5577826ff8d3b4ab41ee4fbc72
StrLen $3 $0
StrCpy $1 0
Loop:
StrCpy $2 $0 "" $1
StrCpy $2 $2 2
detailprint $2
IntOp $1 $1 + 2
IntCmp $1 $3 loop loop 0



This only works properly if strlen is a multiple of 2. Fixing this is left as an exercise for the reader.
