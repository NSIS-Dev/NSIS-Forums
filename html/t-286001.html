kfank
I was searching answer too. Here is some workaround:

Name "Test"
OutFile "Test.exe"

Section
        ;Hide Cancel button
        GetDlgItem $0 $HWNDPARENT 2
        ShowWindow $0 0

        ;Hide Back button
        GetDlgItem $0 $HWNDPARENT 3
        System::Call 'user32::MoveWindow(i $0, i 0, i 0, i 0, i 0, i 1)'
SectionEnd
