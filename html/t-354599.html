!define MUI_WELCOMEPAGE_TITLE_3LINES

actually I guess it was. this code fixed it:

# Subclass label 1006 on Install page
FindWindow $1 "#32770" "" $HWNDPARENT
GetDlgItem $8 $1 1006
GraphicalInstaller::SubclassLabel /NOUNLOAD $8
