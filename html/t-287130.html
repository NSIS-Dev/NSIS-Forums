There's a function in the NSIS wiki to get the steam install path and username.

http://nsis.sourceforge.net/Get_Stea...d_install_path

To make that function better...

Replace this line in GetSteamAccountName function:

IfFileExists "$R0\steamapps\$R2\*.*" done


With this one: 
IfFileExists "$R0\steamapps\$R2\garrysmod\garrysmod\*.*" done


After that change the function returns the first user with garrysmod installed.
