is there a way to set global shortcut keys for installed exe files. for example, lets say we install files: blah.exe, blah.chm & example.blh in $PROGRAMFILES\blah. now, is there a way to make a global shortcut key for file blah.exe so to run it when u press Ctrl+Alt+B for example. :confused:

createshortcut supports hotkey

22nd June 2007 04:21 UTC

thanks man, works great :up:
just for the reference, in our example it would be:

CreateShortCut "$SMPROGRAMS\blah\blah.lnk" "$INSTDIR\blah.exe" "" "" 0 SW_SHOWMINIMIZED alt|control|a
