forums search enhancements
Every time I work with NSIS, I search the forums regularly. There are a couple of small changes to http://forums.winamp.com/search.php that could speed up searching a little bit, which would be a big change for those of us who search a lot.

I doubt that anybody here is a forum admin, but maybe if a lot of people and/or a mod agrees, a request could be submitted to an admin by a mod.

Here are the suggested changes:

- Set focus to the Search By Keyword textbox when the page loads. This should be a 1-liner in JavaScript.

- Create a dropdown in the user options page to select a default forum to search.

With those implemented, users could load the search page, immediately type in their query, and click enter to submit it for (optionally) the only forum they wish to search.

Again, these are very minor enhancements, but over the course of thousands of searches, they can save a lot of time.

Does anyone have their own way to search quicker, or did I overlook something (as usual)?

To search quicker, use the quick search box at the bottom of each forum page. You can also use your browser's quick search function and add a search using http://forums.winamp.com/search.php?...=QUERYGOESHERE.
