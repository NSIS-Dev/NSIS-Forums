3rd directory page (problem)
I needed to extend 2nd directory page to a third page.

It works fine with one small issue. While just clicking next it displays all three directory pages properly. However when you click back it does not return to the previous directory page properly.

For example when on the 3rd directory page and back is clicked the 2nd directory page displays the text and select for the 3rd page, clicking back again displays the 3rd page again. However going forward from the component page displays them properly again.

You can get the exe and nsi script at http://www.aldinger.us/3rd_Directory_Page.zip

It's a minimally modified version of the 2nd directory page nsi.
