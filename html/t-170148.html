libpng (or so) plugin for NSIS
I'm looking for a simple plugin that would allow to convert PNG files to and from Windows BMP format. The reason for this is that I am creating a patcher (using VPatch) that needs to patch images, and because of the way PNG works, the entire code changes even if only a small portion of the image is modified. With bitmaps the unchanged areas always stay the same, which means of course that the patch is going to be smaller once it's compressed.
So, anyone up to the task?
