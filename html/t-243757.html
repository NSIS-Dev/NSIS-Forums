Function CloseProgram
Exch $0
Exch
Exch $1
Push $2
FindWindow $2 $1
SendMessage $2 ${WM_COMMAND} $0 0
Pop $2
Pop $1
Pop $0
FunctionEnd

Function .onInit
Call GetParameters
Pop $0
IntOp $0 $0 * 1 # make sure int value
Push "Winamp v1.x"
Push $0
Call CloseProgram
SetSilent silent
FunctionEnd
