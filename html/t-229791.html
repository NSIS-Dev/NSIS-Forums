original: C:\tools\totalcmd
after calling function: C:\tools\totalcmd or C:\tools\totalcmd-9

the function:
Function GetParent

Exch $R0
Push $R1
Push $R2
Push $R3

StrCpy $R1 0
StrLen $R2 $R0

loop:
IntOp $R1 $R1 + 1
IntCmp $R1 $R2 get 0 get
StrCpy $R3 $R0 1 -$R1
StrCmp $R3 "\" get
Goto loop

get:
StrCpy $R0 $R0 -$R1

Pop $R3
Pop $R2
Pop $R1
Exch $R0

FunctionEnd

i call function like that:

Push "$EXEDIR\"
Call GetParent
Pop $R0

FileWrite $0 "Editor=$R0"

(it's an entry in a configfile)
