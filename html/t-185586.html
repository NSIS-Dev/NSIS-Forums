Place this code into your .onInit function:

  InitPluginsDir
  File /oname=$PLUGINSDIR\splash.bmp "path\to\your\bitmap.bmp"
  advsplash::show 1000 600 400 -1 $PLUGINSDIR\splash
  Pop $0
