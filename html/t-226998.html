NumFields=1

[Field 1]
Type=text
Value=
State=Default Computer Name
Left=

Bottom=

In installer script:

Page Custom Pre Post
...
Function Pre
  ; init dialog
  Pop $R0 ; hwnd to window
  ; show dialog
FunctionEnd

Function Post
  ReadINIStr $R1 <ini file> "Field 1" "State"
  ; $R1 has the value of the computer name entered by user
FunctionEnd
