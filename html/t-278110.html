New plugin - launcher splash
Hi, I modified NewAdvSplash to work better with application launchers.
The problem: with NewAdvSplash, you have to specify fixed time you want it to be displayed. But on different machines the launching time can vary, it's impossible to have one value that's always good.
The solution:
Launcher Splash hides itself when an application you launched creates a window of a specified class.
Get it.
