You could use variables in the caption or any other string and set those variables according to whatever conditions you wish, including the command line.

Var CAPTION
Caption $CAPTION
Function .onInit
${If} $0 == "something"
  StrCpy $CAPTION "some caption"
${EndIf}
FunctionEnd
