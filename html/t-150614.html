!define SF_SELECTED 1

Function myfunc
SectionGetFlags ${secid} $R0
IntOp $R0 $R0 & ${SF_SELECTED}
StrCmp $R0 ${SF_SELECTED} 0 +2
MessageBox MB_OK "Is selected!"
FunctionEnd

That should work.
secid is set like so: Section "section name" secid
