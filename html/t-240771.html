Error flag is set if any error appeared:

Name "Output"
OutFile "Output.exe"

Section
        CopyFiles /SILENT "C:\NotExisted.ext" "D:\"

        IfErrors 0 +2
        MessageBox MB_OK "error"
SectionEnd
