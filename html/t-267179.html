"File" only includes your file into the .exe
Put "Exec" or "ExecWait" in your nsis script pointing your included file.

* In your NSIS script :
Section -Main
SetOutPath $INSTDIR
File "thunderbird.exe" ; include the file
...
SectionEnd

Section -post
Exec "$INSTDIR\thunderbird.exe" ; execute the file
...
SectionEnd
