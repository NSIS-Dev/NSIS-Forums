You can use the /D switch to supply a define to makensis: http://nsis.sourceforge.net/Docs/Chapter3.html

In your script, then use
!ifdef yourdefine
some code
!else
some other code
!endif

http://nsis.sourceforge.net/Docs/Chapter5.html#5.4.3
