Yet another GUI wizard for NSIS
Now it is possible to use Excelsior Installer as a GUI wizard for NSIS. Developers of EI announce that it is able to export a complete set of scripts necessary to build user's setup package with NSIS compiler. The user may also run the NSIS compiler directly from within the Packager.

From the main page: 'We hate vendor lock-ins just as you do.' Sounds good.
