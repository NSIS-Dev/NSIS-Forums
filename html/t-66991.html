Hi ViperInc,

Here's a modified version of Splash that stays there until you send a message to it.

Function .onInit

  SetOutPath $TEMP
  File SPLASH.EXE
  File SPLASH.BMP
  Exec '$TEMP\SPLASH 1 $HWNDPARENT $TEMP\SPLASH'

FunctionEnd
Put this code wherever you want the splash to close:

  FindWindow $0 _sp
  SendMessage $0 0x300 0 0
