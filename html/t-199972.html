Ah right, in that case (if you are using MUI):

!define MUI_PAGE_CUSTOMFUNCTION_LEAVE PromptQLShortcut
!insertmacro MUI_PAGE_STARTMENU ... ...

Function PromptQLShortcut

 MessageBox MB_YESNO|MB_ICONQUESTION ...


FunctionEnd
