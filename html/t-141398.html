Language Files Update
LANGAUGE FILES UPDATE

Please download the latest development version and check the version numbers of your language files

The version number helps you to see whether you have to update your file and what you have to do. You have to make all the changes from the current version of your file to the latest version. Example: if your Modern UI file has version 1.63, you have to make the 1.63-1.64, the 1.64-1.65 changes and the 1.65-1.66 changes.

Note that you always have to download the latest development version before updating your files, because string names or texts might have been changed by NSIS developers or other translators.

You can post the updated files in this topic. If you are not sure whether your language file is correct, please check it against the latest English file.

MODERN UI 1.67

See http://forums.winamp.com/showthread....hreadid=153617

New strings have been added for the new pages. It does not break up scripts, English will be used for missing strings.

The UninstallingText string in the NLF file has also been changed.

OLDER VERSIONS

  • Files with version 1.65 have to be updated. See this page for a list of the changes.
  • Files with version 1.64 have to be updated. Add the LicenseForceSelection texts and modify some other texts:

    Add the following strings:

    MUI_INNERTEXT_LICENSE_BOTTOM_CHECKBOX "If you accept the terms of the agreement, click the check box below. You must accept the agreement to install ${MUI_PRODUCT}."

    MUI_INNERTEXT_LICENSE_BOTTOM_RADIOBUTTONS "If you accept the terms of the agreement, select the first option below. You must accept the agreement to install ${MUI_PRODUCT}."

    Modify the following strings to match the new English texts:

    MUI_INNERTEXT_LICENSE_BOTTOM "If you accept the terms of the agreement, click I Agree to continue. You must accept the agreement to install ${MUI_PRODUCT}."

    MUI_UNTEXT_FINISH_TITLE "Uninstallation Complete"
  • Files with version 1.63 have to be updated. Add the abort texts:

    MUI_TEXT_ABORT_TITLE "Installation Aborted"

    MUI_TEXT_ABORT_SUBTITLE "Setup was not completed successfully."

    MUI_UNTEXT_ABORT_TITLE "Uninstallation Aborted"

    MUI_UNTEXT_ABORT_SUBTITLE "Uninstall was not completed successfully."
NLF LANGUAGE FILES
  • Files with version v6 are up-to-date
  • Files with version v5 have to be updated. Add the new default page texts. See http://cvs.sourceforge.net/cgi-bin/v...1=1.18&r2=1.21
  • Files with version v4 have to be updated. Add the new text for AllowSkipFiles. See http://cvs.sourceforge.net/cgi-bin/v...1=1.17&r2=1.18
  • Files with version v3 have to be updated. Add the new DLL registration, log installation and file size texts. See http://cvs.sourceforge.net/cgi-bin/v...1=1.15&r2=1.16
  • Files with version v2 have to be updated:. Add the LicenseForceSelection texts. See http://cvs.sourceforge.net/cgi-bin/v...1=1.14&r2=1.15
NSIS

The default dialog font is 'MS Shell Dlg'.

Modern UI

The default font for the title on the Welcome/Finish page is 'Verdana', the default font for the header title is 'MS Shell Dlg'.

Change fonts

For non-western languages, these fonts might not work properly (especially on Windows 9x/ME).

If you want to change the dialog font, edit the NLF language file (there is more information in the file). You can also change the codepage.

For the Modern UI, add one or more of the following strings to the language file (see the Modern UI Readme for details).

MUI_FONT_HEADERDER
MUI_FONTSTYLE_HEADER

MUI_FONT_TITLE
MUI_FONTSIZE_TITLE
MUI_FONTSTYLE_TITLE

Please test the fonts with both Windows NT4/2000/XP and Windows 95/98/ME.
