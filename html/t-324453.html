If you want install files of different languages to the same folder, e.g.:

SetOutPath $INSTDIR\Lang
${Select} $LANGUAGE
${Case} ${LANG_ENGLISH}
File .\lang\en.dll
${Case} ${LANG_GERMAN}
File .\lang\de.dll
${Case} ${LANG_FRENCH}
File .\lang\fr.dll
${EndSelect}

If to different directories, e.g.:

${Select} $LANGUAGE
${Case} ${LANG_ENGLISH}
SetOutPath $INSTDIR\en
File .\lang\en.dll
${Case} ${LANG_GERMAN}
SetOutPath $INSTDIR\de
File .\lang\de.dll
${Case} ${LANG_FRENCH}
SetOutPath $INSTDIR\fr
File .\lang\fr.dll
${EndSelect}
