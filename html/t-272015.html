If the name of the file will be static, you could try doing:


Section -
  CopyFiles /SILENT "$INSTDIR\myfiletocheck.ext" "$PLUGINSDIR"
  ClearErrors
  Delete "$INSTDIR\myfiletocheck.ext"
  IfErrors 0 next
    #prompt user to take action
  next:
SectionEnd
