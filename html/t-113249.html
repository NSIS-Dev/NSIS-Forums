VENIS - Visual Editor for NSIS
I have updated Venis, a Visual Enviornment for Nullsoft Install System.

http://www.spaceblue.com/venis/

Venis incorporates syntax highlighting of key words, captured compile results, standard editing features (find/replace/goto) and more.

Version 1.0.4 adds a wordfile with the latest NSIS 2.0a7 key names for syntax highlighting and a new Function List dialog for easy drag-n-drop programming of typical NSIS functions. Each function is enhanced with a wizard to enter function arguments.

Please visit the Venis site and do not hesitate to give me feedback on features/improvements you would like to see. E-mail: angelo@spaceblue.com
