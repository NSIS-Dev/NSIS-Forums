Calling abort will stop the installer:
PHP Code:

Section
SendMessage$HWNDPARENT0x40810;Hacktomovetothenextpage
Abort
SectionEnd
...but, this means the install has been aborted and the only thing the user can do is to close the installer.

If you want to continue doing stuff after the stop, you can't call abort. You can emulate the abort with some ugly code:

PHP Code:
varabortedhack
 !macroBeginAbortableSection
StrCmp$abortedhack""0abortedhack_end
 !macroend
 !macroEndAbortableSection
gotoabortedhack_end
abortedhack_stop:
 SetAutoClosetrue
StrCpy$abortedhack1
abortedhack_end:
!macroend
 !macroStopInstall
gotoabortedhack_stop
 !macroend


Section
 !insertmacroBeginAbortableSection

MessageBoxmb_yesno"Stop?"IDNOnope
 !insertmacroStopInstall
nope:

!insertmacroEndAbortableSection
SectionEnd


Section
 !insertmacroBeginAbortableSection

MessageBoxmb_yesno"Stop?"IDNOnope
 !insertmacroStopInstall
nope:

!insertmacroEndAbortableSection
SectionEnd
