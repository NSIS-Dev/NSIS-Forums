solved
i solved it myself by using this:

loop:
EnumRegKey $1 HKLM "SOFTWARE\JavaSoft\Java Runtime Environment" $0
StrCmp $1 "" done
StrLen $2 "$1"
${If} $2 > 3
${JREFoundArray->Push} "$1"
${EndIf}
IntOp $0 $0 + 1
Goto loop
done:
