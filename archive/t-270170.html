Seems like PSetup.exe expects a URL there and opens it using ShellExecutes with SW_HIDE. Use ShowWindow and BringToFront in .onInit.!include WinMessages.nsh
Function .onInit
ShowWindow $HWNDPARENT ${SW_SHOW}
BringToFront
FunctionEnd
