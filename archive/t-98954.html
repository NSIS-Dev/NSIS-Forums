13th August 2002, 11:00
If it has plug-ins make the data file like this:
Mirror name=http://www.bla.com/prog.exe,http://www.bla.com/plugin.exe

All you need to do is decide on a data structure that fits your needs and deal with it both server side and client side (NSIS).

To pass information from InstallOptions text boxes into one of your PHP scripts is very simple. All you need to do is send your data using GET method (variables in the url eg. http://www.website.com/myscript.php?getplugin1=1&getplugin2=0).
