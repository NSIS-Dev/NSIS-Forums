Every component has section, so add code from this post (http://forums.winamp.com/showthread.php?s=&threadid=229154&highlight=3rd+party) to FLASH' section. Might be usefull to check flash installer presents in $EXEDIR and do not show components page like in this thread (http://forums.winamp.com/showthread.php?s=&threadid=235866) (if no other components). You can also check FLASH is already installed and clear check mark on the components page. You can show message box with warning if FLASH is not installed and user decided to skip it's installation (in the components page leave function) or FLASH package not found in EXEDIR.
