in the last couple of weeks, if added nsis support to several javascript highlighters. embed these in your website and all nsis code will be highlighted like in your favourite text editor.

- Highlight.js (http://nsis.sourceforge.net/Highlight.js)
- Lighter (http://nsis.sourceforge.net/Lighter)
- EnlighterJS (http://nsis.sourceforge.net/EnlighterJS) (a Lighter fork)

personal favourite is highlight, it comes with a great variety of popular colour themes (http://softwaremaniacs.org/media/soft/highlight/test.html).

(the popular pygments (http://pygments.org/) had nsis support for quite some time already)
