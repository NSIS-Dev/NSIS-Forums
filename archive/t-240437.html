kichik
10th March 2006, 14:27
There are a few errors in that System call. You didn't specify a type for the first two arguments and you forgot to close both NSIS_MAX_STRLEN uses. You also set the wrong type for the last argument. It should be a pointer to an integer, not an integer. The call should be:System::Call 'winspool.drv::GetPrinterDriverDirectory(i 0, i 0, i 1, t .r1, i ${NSIS_MAX_STRLEN}, *i)'
