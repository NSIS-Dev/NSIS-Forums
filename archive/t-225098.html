To read a registry string, you can use "ReadRegStr" or "ReadRegDWORD" instructions. If those don't work for you, you can use the Registry plug-in (http://nsis.sourceforge.net/wiki/Registry_plugin) by Instructor.

To compare values, you can use either "StrCmp" or "IntCmp" instructions. If you need to compare versions, you can use one of the functions in the Version Manipulation Functions category (http://nsis.sourceforge.net/wiki/Category:Version_Manipulation_Functions).

Well, now, to have a page with "re-install" or "repair", and "remove" options, you need to create a new InstallOptions page.

Also, you could use some other UI's available for NSIS, like for example: Experience UI (http://nsis.sourceforge.net/wiki/Experience_UI).

First of all clear your duplicate posts.
If your reg key was found you can hide installer and run uninstaller. After uninstall bring to front installer. See example folder (makensis.nsi).
Or you can (if you want) change set of sections with installer's components page. This (http://nsis.sourceforge.net/wiki/Add/Remove_Functionality) script will help you add/remove components.
Edit:
To skip repair/uninstall (custom) page - in pre-function use Abort.
