19th February 2010, 11:09
You could pack your files inside a LZMA 7ZiP archive and use the 7Z plugin (http://nsis.sourceforge.net/Nsis7z_plug-in).

You could then calculate the percentage by using the given variables. Easiest way ... at least for me because i use this plugin in almost all my installers together with the Math plugin (http://nsis.sourceforge.net/Math_plug-in).

But there are 2 things about the 7z plugin you should consider:
- no split archives,
- no archives >1.2GB (may/will result in an negative amount of data for the whole size [only happend once to me])
- only LZMA, no LZMA2.
