yes.
write the text-strings with "MUI_INSTALLOPTIONS_WRITE "ioPage.ini" "Field 2" "State" $(yourlabeltext)".
it's recommended to do so in the enter-function Page custom enterfunction leavefunction
...
Function enterfunction
#writing ...
FunctionEnd
see the docs for further information about installoptions, the wirte-macros in MUI and custom pages.
