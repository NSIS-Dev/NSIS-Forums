If you're only using the UserInfo plugin to get the operating system then it'd be better to use the GetWindowsVersion function (http://nsis.sourceforge.net/Docs/AppendixC.html#C.5).

As for not displaying the page, just jump over the InstallOptions call or call Abort before displaying it.
