SetOutPath <outpath>

Sets the output path ($OUTDIR) and creates it (recursively
if necessary), if it does not exist. Must be a full pathname,
usually is just $INSTDIR (you can specify $INSTDIR if you
are lazy with a single "-").

SetOutPath $INSTDIR
File program.exe



SetOutPath <a-dir>
file a.ext

SetOutPath <b-dir>
file b.ext
