nsL Assembler died when I finished my Uni degree so here is the next best thing. A small C plug-in (6KB) which extends the functionality of NSIS to support C-style expressions. They can be used in an ${If} like so:${If} ${Expr} `(R0 == 9 || r0 == 3) && (R0 == 0 || !r2)`
...Or they can be evaluated with their result written to a variable:${ExprEval} `(r3 + 20) / 3 * -2` $R0http://nsis.sourceforge.net/NsExpr_plug-in

See Readme.txt in the zip file for available operators. May need some further testing.
