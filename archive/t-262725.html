I'm not sure if I understood the right thing, if you only read a single line that contains only e.g. $DESKTOP this might help:
;Read the string in $0
${TrimNewLines} $0 $0
DetailPrint $0
StrCmp $0 '$$DESKTOP' 0 end
StrCpy $0 '$DESKTOP'
end:
detailprint '$0'
