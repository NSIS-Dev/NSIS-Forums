UnzipDLL is a extension DLL for NSIS. It can unzip all files from zip files. It is especially useful in combination with NSISdl so that you don't have to download large files uncompressed.

Know limitations:
Unfortunately this unzip library can't decompress .zip files with subdirectories. :cry:
