I thought Eugene234 was referring to the CRC check message. If that is what he is seeing, then he cannot hide it in .onInit because the nsi script has not started executing yet. It can be disabled from the command line with /NCRC or a script command CRCCheck off (http://nsis.sourceforge.net/Docs/Chapter4.html#4.8.1.12)
