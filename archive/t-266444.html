You can calculate number of selected sections and compare result with 1:Function .onSelChange

StrCpy $R1 0
SectionGetFlags ${Section1} $R0
IntOp $R0 $R0 & ${SF_SELECTED}
IntCmp $R0 ${SF_SELECTED} 0 +2
IntOp $R1 $R1 + 1
SectionGetFlags ${Section2} $R0
IntOp $R0 $R0 & ${SF_SELECTED}
IntCmp $R0 ${SF_SELECTED} 0 +2
IntOp $R1 $R1 + 1
SectionGetFlags ${Section3} $R0
IntOp $R0 $R0 & ${SF_SELECTED}
IntCmp $R0 ${SF_SELECTED} 0 +2
IntOp $R1 $R1 + 1
IntCmp $R1 1 0 skip skip
; some code
skip:
MessageBox MB_OK "Selected $R1"

FunctionEnd
