From which installer? Are you running another installer from your own installer or are you talking of your own installer?

If it's another installer, use ExecWait and pass a variable to be filled with the return code as the second parameter.ExecWait "another installer.exe" $0If it's your installer, run it in a cmd.exe with start /wait and use %ERRORLEVEL% later.
