Push "Installer"
Push "sta"
Call StrStr
Pop $0
StrCmp $0 "" +2 +6

Using StrStr (off NSIS documentation) the output will be "" if "sta" was not found within it. Otherwise, the output will be "staller".
I have used StrStr a LOT throughout my NSIS aps.
