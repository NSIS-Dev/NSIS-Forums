If I can get a handle on the label, is there a way to change the text so give the user some kind of indication of what's happening? Several example scripts are shipped with NSIS. The makensis.nsi script (in the "Examples" folder) shows how to do this. Here is a fragment from that file:
SetDetailsPrint textonly
DetailPrint "Installing NSIS Core Files..."
SetDetailsPrint listonlyThe NSIS installer creates a Start Menu shortcut to the "Examples" folder
