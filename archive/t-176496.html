hi

use this:

Function .onInit
InitPluginsDir
File /oname=$PLUGINSDIR\yourBmp.bmp "yourBmp.bmp"
WriteINIStr "$PLUGINSDIR\YourINI.ini" "Field 8" "Text" "$PLUGINSDIR\yourBmp.bmp"
FunctionEnd

you have to edit:
yourBmp.bmp
YourINI.ini
Field 8

not tested but it should work fine
