kichik
6th January 2006, 11:37
You can set the z-order of the controls using the System plug-in. The z-order sets the tab order of the controls. For example, to reverse the tab order of Field 6, use:!define SWP_NOSIZE 0x0001
!define SWP_NOMOVE 0x0002
!define SWP_NOACTIVATE 0x0010

ReadINISTr $0 "$PLUGINSDIR\test.ini" "Field 6" "HWND"
ReadINISTr $1 "$PLUGINSDIR\test.ini" "Field 6" "HWND2"
System::Call "user32::SetWindowPos(i $0, i $1, i 0, i 0, i 0, \
i 0, i ${SWP_NOSIZE}|${SWP_NOMOVE}|${SWP_NOACTIVATE})"
