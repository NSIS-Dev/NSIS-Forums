None of those affect an external program. You'll have to use built-in registry commands. You can use reg2nsis (http://nsis.sourceforge.net/Reg2Nsis_-_convert_registry_info_into_NSIS_commands) to convert the .reg file automatically.

It's not an internal program, but an internal command that runs in the same process. You can only affect your own process with the macros in x64.nsh.
