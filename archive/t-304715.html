Use a ${Switch} or ${If} with $LANGUAGE and extract your different files depending on the value of $LANGUAGE using File with the /oname switch.

e.g.

${If} $LANGUAGE == 1033
File /oname=language.config languageEN.config
${ElseIf} ...
...
${EndIf}
