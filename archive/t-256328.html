At the risk of sounding totally clueless (which may not be far from the truth):

When using the InstallLib macro, how do I tell which libtype option is needed?
kichik
29th September 2006, 12:15
DLLs requiring registration are exporting the DllRegisterServer function. You can check which functions DLLs export using Dependency Walker (http://www.dependencywalker.com/). You could also try both and see if you get an error saying DllRegisterServer can't be found.

kichik
29th September 2006, 15:27
TLB option is only used for type libraries. Those don't have an exported function, but instead, that exported function you found is used to register them.
