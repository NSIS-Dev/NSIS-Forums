JasonFriday13
1st August 2006, 22:18
Like this:
Function .oninit
;For MUI users:
!insertmacro MUI_INSTALLOPTIONS_EXTRACT "yourpagefile.ini"

;For normal users:
File /oname=$PLUGINSDIR\yourpagefile.ini "yourpagefile.ini"
FunctionEnd
