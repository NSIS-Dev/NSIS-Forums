Note that if you use different icons for installer and uninstaller the file size and structure of the icons has to match otherwise the build of your installer will fail.What this means is that if your installer icon file contains a 16x16 icon in 16 colors and a 32x32 icon in 256 colors then your uninstaller icon file must also contain a 16x16 icon in 16 colors and a 32x32 icon in 256 colors.

The two icon files must contain the same number of icons, in the same sizes and with the same number of colors (and I think the order within the file has to be the same too).
