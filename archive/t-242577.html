You should set the SF_RO flag. You can set it on runtime using the macros in Sections.nsh.!include Sections.nsh
#...
!insertmaro SetSectionFlag ${some_section} ${SF_RO}
!insertmaro ClearSectionFlag ${some_section} ${SF_RO}
