That's not exactly what silent means on NSIS. What you want is an installer with nothing but the instfiles page. Edit Contrib\UIs\default.exe with a resource editor such as Resource Hacker and have your script use the new version with ChangeUI. When editing, add a bitmap place holder and show the bitmap in it using LoadImage and WM_SETIMAGE. A complete example can be found in the Wiki (http://nsis.sourceforge.net/ModernUI_Mod_to_Display_Images_while_installing_files).

If you want to show just one picture, you can skip the loading part on the script and simply embed it as a resource. The problem is, ChangeUI only changes the dialog resources. To add a bitmap resource, you'd have to call Resource Hacker using !packhdr.
