Use FileOpen, FileRead to read a file line by line.
There are various string functions on the archive which will allow you to get access to the string chunk you want.

Something like this:

Function GetCodeThingy
Push $R0
Push $R1
Push $R2
FileOpen $R0 "path_to_file.ext"
readTop:
ClearErrors
FileRead $R0 $R1
IfErrors readDone
StrLen $R2 $R1
StrCpy $R2 $R1 21 -$R2
StrCmp $R2 "CsComm ConnectionPool" 0 readTop
Push $R1
Call GetLastStrPart
Pop $R1
Push $R1
Call TrimNewLines
Pop $R1
readDone:
FileClose $R0
StrCpy $R0 $R1
Pop $R2
Pop $R1
Exch $R0
FunctionEnd

Function TrimNewlines
Exch $R0
Push $R1
Push $R2
StrCpy $R1 0

loop:
IntOp $R1 $R1 - 1
StrCpy $R2 $R0 1 $R1
StrCmp $R2 "$\r" loop
StrCmp $R2 "$\n" loop

IntOp $R1 $R1 + 1
IntCmp $R1 0 no_trim_needed
StrCpy $R0 $R0 $R1

no_trim_needed:
Pop $R2
Pop $R1
Exch $R0
FunctionEnd

Function GetLastStrPart
Exch $0
Push $1
Push $2
StrCpy $1 0
loop:
IntOp $1 $1 - 1
StrCpy $2 $0 1 $1
StrCmp $2 "" exit2
StrCmp $2 " " exit1
Goto loop
exit1:
IntOp $1 $1 + 1
StrCpy $0 $0 "" $1
exit2:
Pop $2
Pop $1
Exch $0
FunctionEnd


This should work.
Use:
[code]
Call GetCodeThingy
Pop $R0
