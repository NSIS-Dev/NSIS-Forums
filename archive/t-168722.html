This is how

Section

File /a "THE DESTINATION OF THE FILE"

SectionEnd
; NOTE : MUST BE IN SECTION

Example

File = kichik.exe
Location of FILE = C:

Then you must use
File /a "C:\kichik.exe"

Dont forget to put it in a section!!!
