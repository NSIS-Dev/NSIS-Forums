You can use FileOpen (http://nsis.sf.net/Docs/Chapter4.html#4.9.5.2), FileWrite (http://nsis.sf.net/Docs/Chapter4.html#4.9.5.6), and FileClose (http://nsis.sf.net/Docs/Chapter4.html#4.9.5.1) to create a BAT file. As for asking the user for parameters, I suspect you'll need to use InstallOptions (http://nsis.sf.net/Contrib/InstallOptions/Readme.html) to create a custom page, but I haven't had much experience doing that, and someone else will likely be able to help you with that.
