Use the SectionFlagIsSet macro in Include\Sections.nsh (!include "Sections.nsh") along with the SF_SELECTED flag to check if a certain section is selected. For example:!insertmacro SectionFlagIsSet ${section1} ${SF_SELECTED} selected notselected
selected:
MessageBox MB_OK "section 1 is selected"
Goto done
notselected:
MessageBox MB_OK "section 1 is NOT selected"
done:
