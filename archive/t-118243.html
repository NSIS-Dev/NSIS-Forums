Use $SYSDIR to get the system directory. For example:

MessageBox MB_OK "System directory is '$SYSDIR'"
# or
StrCmp $SYSDIR "C:\\WinNT\\System" NT NotNT

NotNT:
# do not NT stuff
Goto done

NT:
# do NT stuff

done:


If you do this comparsion to test if you are installing on Windows NT you better use GetWindowsVersion (http://nsis.sourceforge.net/Docs/AppendixB.html#B.5).
