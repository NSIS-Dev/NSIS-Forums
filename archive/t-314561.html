Similar to MultiCmd (http://nsis.sourceforge.net/MultiCmd:_One_NSIS_command_instead_of_multiple), this macro allows the programmer to write a multiple line statement in a single line of code. The primary difference in this version is that the process is handled entirely by the compiler and dose not rely on an external/pre-compiled executable. Thus allowing for increased portability.

Let me know if there are any bugs. :D
