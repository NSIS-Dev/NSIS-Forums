I found the post, and I put the code here!


!define SF_SELECTED 1

Page custom CustomPageName

Function CustomPageName
SectionGetFlags $0 ${SectionDefineName}
StrCmp $0 ${SF_SELECTED} 0 skipPage
# show page here ...
# InstallOptions::dialog....
skipPage:
FunctionEnd
