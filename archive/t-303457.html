So,

something like:


StrCpy $1 "A String With Spaces"
System::Call 'MyDLL::MyFunc(t)i(r1).r0? c'


Results in MyFunc receiving a string buffer that starts with \0 i.e. an empty string.

If I do this instead:


StrCpy $1 "AStringWithoutSpaces"
System::Call 'MyDLL::MyFunc(t)i(r1).r0? c'


... then it works.
