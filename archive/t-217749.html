If you want create new XML file (Docs (http://nsis.sourceforge.net/Docs/Chapter4.html#4.9.5)):
FileOpen $0 "C:\file.xml" w
FileWrite $0 '<username> "value to be read from custom page"</username>$\r$\n'
FileWrite $0 '<password> "value to be read from custom page"</password>$\r$\n'
FileClose $0

If you want modify existing XML you could try this:
Name "Output"
OutFile "Output.exe"

!include "TextFunc.nsh"
!insertmacro ConfigWrite

Section
StrCpy $0 'password'
StrCpy $1 ' "value to be read from custom page" '

${ConfigWrite} "C:\file.xml" "<$0>" "$1</$0>" $R0

IfErrors 0 +2
MessageBoX MB_OK "error" IDOK +2
MessageBoX MB_OK "$R0"
SectionEndScript use header (http://forums.winamp.com/attachment.php?s=&postid=1680197)

Note:
This script for complite line change,<pane>
<guid>123</guid>
</pane> NOT for<pane>
<guid>123</guid></pane>
