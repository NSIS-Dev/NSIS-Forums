Use the command SetDetailsPrint: http://nsis.sourceforge.net/Docs/Chapter4.html#4.9.14.14

The makensis.nsi script in the NSIS\Examples folder shows how to display more meaningful messages; here is a fragment from that file SetDetailsPrint textonly
DetailPrint "Installing NSIS Core Files..."
SetDetailsPrint listonly
