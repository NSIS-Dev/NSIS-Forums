You can edit the UI (Contrib\UIs\modern.exe) using Resource Hacker (http://www.angusj.com/resourcehacker/). You can tell the MUI to use your UI using the MUI_UI define. If you're not using the MUI, you can edit Contrib\UIs\default.exe and use ChangeUI to use your new UI.
