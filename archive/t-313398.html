I am using the following code to disable next button.


Page custom InstallTypeStart InstallTypeEnd

Function InstallTypeStart
GetDlgItem $0 $HWNDPARENT 1
EnableWindow $0 0

!insertmacro MUI_INSTALLOPTIONS_DISPLAY "installtype.ini"
FunctionEnd
