WMI has a COM API (http://msdn2.microsoft.com/en-us/library/aa389276.aspx) which you can use with the System plug-in. However, using COM with System isn't that comfortable and best suited for small-sized scripts. You have to look for the ordinal of every method in the header files before you can call it.
