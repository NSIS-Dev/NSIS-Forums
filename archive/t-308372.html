Hi, nothing changed in the plugin.

Be aware that plugins are removed when updating NSIS (if you choosed to uninstall the old version, which is default).

Check the %PROGRAMFILES%\NSIS\Plugins folder if ExecCmd.dll is there.
If not, you have to install the ExecCmd plugin again.
