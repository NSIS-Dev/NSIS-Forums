You can get it from CVS:

http://cvs.sourceforge.net/viewcvs.py/nsis/NSIS/Contrib/UIs/Attic/modern2.exe?rev=1.20&view=log

But why? It's not used by any modern scripts. If you need it to compile a script, you'd probably want to use an older version of NSIS which comes bundled with it. It was removed after 2.0b2, so it should still be present in 2.0b2 and below.
