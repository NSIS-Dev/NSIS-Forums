It is probably not a good idea, since you custom page's dimensions will constrain you from showing all the files (if there are many of them). You can use InstallOptionsEx (http://nsis.sourceforge.net/InstallOptionsEx) to create a custom page that has a DropList control showing the files in a certain directory.
