Hi, this is bramoin again. I found the answer to this question i had and since i had to dig for it and finally asked my colleage for it. Here is the answer for anyone who will run into this:


If you already have an nsi script loaded into the Eclipse editor before you load the plug in you may have to exit and restart, I know I did but I don't recall exactly why I needed to.
Use the Help -> Install New Software menu to open the dialog.
Click the "Add" button
For description type "EclipseNSIS"
For address type "http://eclipsensis.sf.net/update"
Click OK, the archive will be queried and you should see a list of components.
Click "Select All" and install them.
