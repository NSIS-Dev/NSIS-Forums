Originally posted by katipunero
...I can use SetShellVarContext to do this.

But I have a problem. Will this also be applicable to ReadRegStr when I read a value in HKCU?No, it won't. You'll have to write two code paths, one for HKLM and one for HKCU. You can also use a simple macro like:!macro ReadRegStr var key value
StrCmp $ALLORCURRENT all 0 +3 # replace with real check
ReadRegStr "${var}" HKLM "${key}" "${value}"
Goto +2
ReadRegStr "${var}" HKCU "${key}" "${value}"
!macroend
