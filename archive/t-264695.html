You could create a macro of each command, which would log the event.

!macro SetOutPath _PATH
; logging done here
SetOutPath `${_PATH}`
!macroend
!define SetOutPath `!insertmacro SetOutPath`
...
Section Blah
${SetOutPath} "$INSTDIR"
...
SectionEnd
