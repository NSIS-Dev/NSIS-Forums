I translated a bunch of API calls to system plugin commands which can be found on the wiki (http://nsis.sourceforge.net/User_Management_using_API_calls).

If you have already created the user, you can call NetUserSetInfo (http://msdn2.microsoft.com/en-us/library/aa370659.aspx) from advapi32, using a USER_INFO_1008 (http://msdn2.microsoft.com/en-us/library/aa370968.aspx) structure which contains the UF_DONT_EXPIRE_PASSWD attribute.
Hope this helps
