I have already solved it using this "case" statement:
SetOutPath "$INSTDIR\$Condition"
${If} $Condition == "11"
File "${SourceDir}\11\a.txt"
${ElseIf} $Condition == "99"
File "${SourceDir}\99\a.txt"
${EndIf}
This is easy enough. Thank you.
