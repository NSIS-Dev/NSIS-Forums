The numbers shown in MakeLangId.exe are created using the MAKELANGID (http://msdn.microsoft.com/library/en-us/intl/nls_97vo.asp) macro defined in the Windows API header files. Its source code is now in CVS and is also attached. I don't know why I haven't uploaded it before.

You could try using the code page from that website. If Windows supports the Khmer code page, it should work.

For switching without rebooting you can try Nolybab (http://nsis.sourceforge.net/archive/viewpage.php?pageid=346).
