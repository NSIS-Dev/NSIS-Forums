1.98 is the last stable release by Justin.
2.0a0 is the last alpha release by Justin.
These can be found on the official NSIS page at Nullsoft (http://www.nullsoft.com/free/nsis).

1.99 is a modification by forum user KiCHiK.
2.0a1 is a modification by forum user Rainwater.
2.0a2 is the combination of 1.99 and 2.a01 plus some additional features, by KiCHiK.

The 1.99 mod is hosted on my archive here (http://www.clantpa.co.uk/nsis/wiki/index.php/Nsis199).

I have a page for NSIS 2.0a0 on my archive here (http://www.clantpa.co.uk/nsis/wiki/index.php/Nsis20a1) and the file available for download but that is not it's official home, I'd have to dig through the archives of this forum to find where Rainwater made it available to work that out.

The 2.0a2 mod has it's own area on Source Forge here (http://sourceforge.net/projects/nsis2k/).

Last I heard Justin was going to be away for a while, so the official position on NSIS I guess is that it's sleeping for now, bar user developments.
