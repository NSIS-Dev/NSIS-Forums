You can use the Math plug-in to for floating point math.StrCpy $0 31 # number of files
Math::Script "R0 = 100.0/r0"
DetailPrint $R0 # file weight
StrCpy $0 23 # file number one
Math::Script "r0 = i(R0*r0)"
DetailPrint $0
