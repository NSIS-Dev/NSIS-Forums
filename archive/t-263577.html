On the 2nd custom page create function, read the state of 1st page radio buttons.
If radio button "no" is 1 (checked) then skip the page.
e.g.
ReadINIstr $1 "ini_path\ini_name" "field_num" "state"
${if} $1 == 1
Abort
${endif}
