There's an if command (http://nsis.sourceforge.net/Docs/Chapter2.html#2.3.5.1). Your code should up looking something like this:!include LogicLib.nsh
# ...
${If} $LANGUAGE == ${LANG_SLOVAK}
# something
${EndIf}
