Any idea how to run installation of SQL Server Reporting Services with nsis!?!?!?

Insert the Microsoft SQL Server 2005 product CD into the CD Drive.

Click Start, point to Settings, and then click Control Panel.

In Control Panel, double-click Add or Remove Programs.

In Add or Remove Programs, click Microsoft SQL Server 2005, and then click Change.

On the Component Selection page, expand SQL Server 2005 instances, select MSSQLSERVER, and then click Next.

On the Feature Maintenance page, select Reporting Services, and then click Next.

The Microsoft SQL Server Installation Wizard starts.

On the Welcome to the Microsoft SQL Server Installation Wizard page, click Next.

On the System Configuration Check page, click Next.

On the Change or Remove Instance page, click Change Installed Components.

On the Feature Selection page, make sure that Reporting Services will be installed on the local hard disk drive, and then click Next.

On the Ready to Update page, click Install.

On the Setup Progress page, click Next.

On the Completing Microsoft SQL Server 2005 Setup page, click Finish.
