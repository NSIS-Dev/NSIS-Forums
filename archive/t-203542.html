As for finding the cookie, you can use FindFirst, FindNext etcPush $R0
FindFirst $R0 $R1 "$COOKIES\*.*"
Loop:
IfErrors Done
StrCpy $R2 $R1 "" -15
StrCmp $R2 "@website[1].txt" 0 NextCookie

FileOpen $R2 "$COOKIES\$R1" "r"
CookieLoop:
ClearErrors
FileRead $R2 $R3
IfErrors CookieDone

## Find cookie id
Push $R3
Call TrimNewLines
Pop $R3
StrCmp $R3 "mycookieID" 0 CookieLoop

## Get value
FileRead $R2 $R3
Push $R3
Call TrimNewLines
Pop $R3

WriteRegStr HKCU "Software\myapp" "Cookie" "$R3"

CookieDone:
FileClose $R2
Goto Done

NextCookie:
ClearErrors
FindNext $R0 $R1
Goto Loop
Done:
FindClose $R0Note that this isn't tested!

Edit: You need the TrimNewLines function in your script.
