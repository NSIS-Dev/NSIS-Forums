It should filter out paste events as well. ES_NUMBER style is used for it, which apparently doesn't care for pastes. Even MSDN says that.Allows only digits to be entered into the edit control. Note that, even with this set, it is still possible to paste non-digits into the edit control.Please submit a bug report about this.

For now, you can manually check the field's value in the leave function of the page.
