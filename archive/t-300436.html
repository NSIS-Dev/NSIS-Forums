A new and improved version of WinVer.nsh is coming in the next version of NSIS (2.42). It supports Windows Server 2008, fixes Windows XP x64 identification, resolves ambiguity of Windows 95 and NT4; and adds a couple more features. It's based on a patch by Andres.

I've also beefed up its tests so I don't have to waste my time testing on numerous virtual machines but quickly test it on my main machine. To do this, I've written a script that fakes each configuration I could get my hands on. If you're interested, you can read more about it in my blog (http://kichik.net/2008/11/29/pragmatic-variant/).

My collection of Windows versions is extensive, but doesn't cover everything. And this is where you come in. If you have any configuration not listed in the test script (http://nsis.svn.sourceforge.net/viewvc/nsis/NSIS/trunk/Source/Tests/winver.nsi?view=markup), please run the attached information collector and post the results here.
