View the SetOverWrite (http://nsis.sourceforge.net/Docs/Chapter4.html#4.8.2.8) documentation. It tells the File instruction how to proceed in cases where it can't immediately overwrite an existing file.

edit: If you have a file that is in-use at install time, you can send it a close message or kill its process if it is an EXE, or use the InstallLib macro if it is a DLL.
