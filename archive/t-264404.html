$INSTDIR is not saved anywhere. The uninstaller uses the directory from which is was executed as $INSTDIR. If you put it in a subdirectory, you can use un.GetParent (http://nsis.sourceforge.net/Docs/AppendixE.html#E.1.15) in the uninstaller to get the parent directory.
