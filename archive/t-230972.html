I assume you are using the same rtdesign library as HMNISEdit.
Some bugs:
[list=1]
Doesn't allow editing in text mode.
Cut/paste don't work in text mode.
GroupBox should be transparent.
Negative coordinates are not supported properly.
Units should be dialog units- not pixels.
File extension is .ini, not .nsf.
Cannot save to an INI file.
Cannot open a preexisting INI file.
DropList, ComboBox, ListBox do not use the Text field. They use the State field.
Cannot set dialog level attributes (e.g., CancelEnabled, CancelShow, RTL, Rect, etc).
Cannot figure out how to create helplines.
Z-ordering of controls is reversed.
Link, FileRequest and DirRequest controls are missing.
DropList should not be editable in Runtime mode (i.e., typing should not be allowed).
[/list=1]

These would be nice to haves:
[list=1]
Text control should not allow me to type in letters if ONLY_NUMBERS flag is set.
Listbox should not allow State field to be set to values not specified in ListItems field.
Listbox should not allow State field to be set for multiple values if MULTISELECT or EXTENDEDSELECT flags are specified.
DropList should not allow State field to be set to values not specified in ListItems field.
Password control should show *** instead of actual text.
Controls should enforce MaxLen, MinLen field settings.
CheckBox, RadioButton should accept only 1 or 0 in the State field.
[/list=1]

If I find any other issues I will post them here.
