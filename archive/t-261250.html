I am pretty sure that what you want to do can be achieved just by changing the registry value. As long as the user who is running your app has admin privileges this should work:WriteRegDWORD HKLM "SYSTEM\CurrentControlSet\Control\Lsa" "forceguest" 0Note however that it is a DWORD that you want to change and not a string value ;)
