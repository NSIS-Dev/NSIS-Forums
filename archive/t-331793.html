Anyway, for 2000+ (and maybe NT4):

@echo off
call setup.exe&&(
echo ==0
) || (
echo !=0
)

Or the old school way: (Did not actually test on old systems)
@echo off
call setup.exe
if ERRORLEVEL 1 goto failed
echo ==0
goto end
:failed
echo !=0
:end
