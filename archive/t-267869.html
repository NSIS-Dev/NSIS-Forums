Provided that you use the function from this wiki page,

http://nsis.sourceforge.net/IsUserAdmin

!include "LogicLib.nsh"

Function .onInit
Call IsUserAdmin
Pop $R0
${if} $R0 == "false"
MessageBox MB_OK|MB_ICONSTOP "Installing my application requires admin rights"
Abort
${EndIf}
FunctionEnd
