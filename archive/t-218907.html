I think you should check my wiki page:

http://nsis.sourceforge.net/wiki/index.php?title=InstallOptionsEx

Also, the current development of the next version will include:
- ListView controls.
- RefField, StateImageList, SmallImageList, LargeImageList INI value names.
- COLORREQUEST and FONTREQUEST flags for Link and Button controls.
- "State" INI value name for Animation controls: Specifies the number of times the animation is repeated from the "MinLen" to the "MaxLen" point. If more than 0, the animation will repeat that number of times. If 0, the animation doesn't get played when the dialog is shown. If -1, the number of loops is unlimited.

Default Value: -1
