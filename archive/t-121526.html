Assuming you do want to set the default language for the LangDLL dialog copy ${LANG_ENGLISH} into $LANGUAGE using StrCpy in .onInit:
Funciton .onInit
# LangDLL stuff
StrCpy $LANGUAGE ${LANG_ENGLISH}
FunctionEnd
