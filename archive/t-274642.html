nsExec normally should present in NSIS/plugins folder.
ExecDosDetailPrint "Running netstat -a"
ExecDos::exec "netstat -a" "" "$EXEDIR\netstat.txt"
Pop $0
MessageBox MB_OK "Exit code $0"You should set correct path for output file and can skip MessageBox.
