Full working example code is below. Command line to run the program:Test.exe /S /D=C:\Program FilesThe example:OutFile Test.exe

Function .onInit
IfSilent 0 +3
MessageBox MB_OK "Silent. Install dir: $INSTDIR"
Goto +2
MessageBox MB_OK "Not silent."
FunctionEnd

Section
SectionEndMessageBox output:Silent. Install dir: C:\Program Files
