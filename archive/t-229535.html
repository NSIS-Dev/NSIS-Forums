The error flag will be set if RegDll fails. There won't be any error messages unless you add one.

ClearErrors
RegDll "$SYSDIR\MSFLXGRD.OCX"
IfErrors 0 +2
MessageBox MB_OK "Failed to register MSFLXGRD.OCX!"
