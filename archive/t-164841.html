'SetCompress off' does not have to apply to all files. Just set it before the CAB files and use 'SetCompress auto' afterwards.

This will only work if you are not using solid compression. ZLIB does not use solid compression by default. If you are using BZIP2 or LZMA see http://nsis.sourceforge.net/site/Special_Builds.30.0.html
