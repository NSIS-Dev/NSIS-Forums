Most plug-ins are licensed under the zlib license (http://nsis.sourceforge.net/License) which allows you to modify and use them as long as you don't claim to have written them.

A basic plug-in skeleton in C, ASM and Delphi can be found in Examples\Plugin. The source code for all of the plug-ins distributed with NSIS come in the source code package (http://prdownloads.sourceforge.net/nsis/nsis-2.28-src.tar.bz2?download).
