Use FindFirst, FindNext and FindClose to search a given drive. Take this (http://nsis.sourceforge.net/archive/viewpage.php?pageid=58) archive page for example. Instead of using SetFileAttributes use StrCmp to compare the file name to the file name you're looking for.
