I guess you would have to do it manually:

SetOutPath "$INSTDIR\a\b"
File "a\b\*.*"

SetOutPath "$INSTDIR\a\c"
File /x bob.txt "a\c\*.*"

SetOutPath "$INSTDIR\a\d"
File "a\d\*.*"
