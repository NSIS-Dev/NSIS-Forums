This might work: (try it and see as I don't have a 64-bit computer available at the moment.)!include x64.nsh
...

${DisableX64FSRedirection}
ExecWait 'regedit.exe /s "C:\registry\regfile01.reg"'
ExecWait 'regedit.exe /s "C:\registry\regfile02.reg"'
${EnableX64FSRedirection}The only caveat is that it's not clear to me if this affects REGISTRY functions. (There's also a chance it may not work on ExecWait or Exec.)

The macro just calls the system.dll to call the API function Wow64EnableWow64FsRedirection (http://msdn.microsoft.com/en-us/library/aa365744(VS.85).aspx). You might get more info by searching MSDN.
