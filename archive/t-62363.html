Where should I search for a solution?
Some vital information where to look for a possible solution for your problem.

First of all, the NSIS documentation (http://www.nullsoft.com/free/nsis/makensis.htm).
Do you want some special script (http://www.nullsoft.com/free/nsis/functions.htm)?
More scripts and ExtensionDLLs in the NSIS Archive (http://www.clantpa.co.uk/nsis).
Did you search (http://forums.winamp.com/search.php) the NSIS Forum?
Still no solution? Then post (http://forums.winamp.com/newthread.php?s=&action=newthread&forumid=65) your question at the NSIS Forum. Maybe other NSIS users can help you out.

Please first search the Forum before posting a question to avoid as much as possible duplicate questions.
