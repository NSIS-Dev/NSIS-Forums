You can format at the command line, too:# $USB is the drive letter, $USBVolName is the new name we want to assign
ExpandEnvStrings $0 %COMSPEC%
nsExec::Exec `"$0" /c echo. | Format $USB /Q /X /V:$USBVolName`
Pop $0 ; get the result code (0 = good) The "echo. |" sends an Enter key to the prompt it asks.
