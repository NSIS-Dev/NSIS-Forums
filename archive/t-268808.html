How about creating a header like,
!ifdef ordernumber1
file "c:\dir1\file1"
file "c:\dir1\file2"
!endif

!ifdef ordernumber2
file "c:\dir2\file1"
file "c:\dir2\file2"
!endif

and call the compiler with /D.

You may even automate the creation of the header, see below,
