No, it is not possible to query an Access database from NSIS, or a plugin.
And it is also not possible to do it by running MSAccess with commandline parameters (or you have to make a macro in your access database to do it, see http://support.microsoft.com/kb/209207)
You can do it however with a VBScript, see http://www.visualbasicscript.com/Query-Access-Database-with-VBScript-m59038.aspx.
