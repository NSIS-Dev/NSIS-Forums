You shouldn't declare $WINDIR. It's given.

Just do:
function .onInit
StrCpy $0 $WINDIR 2
; Now $0 is the system drive
StrCpy $INSTDIR "$0\MyApp"
; this is to set the default install dir
; to the right choice.
FunctionEnd
