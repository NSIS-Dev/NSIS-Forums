I'm happy to announce the release of a new GUI for creating NSIS scripts: Atomic GUI for NSIS.

We've actually been using this in-house for our software for a while, but it occurred to us to release it to the general public (with due warnings).

It is meant for use with NSIS2.0b and the "classic" interface (it does not currently support the "Modern UI"), with or without the background gradient. Putting files in the project is all drag & drop, and it has a reasonably nifty shortcut creating gizmo. Source code is included, and a forum for support/suggestions/feedback is ready.

If you're interested, it resides at Atomic Monks (http://www.atomicmonks.com).
