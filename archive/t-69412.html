2) ExecWait open "$INSTDIR\MyApp.exe"

24th December 2001, 11:35
ExecShell open "$INSTDIR\MyApp.exe"
Exec $INSTDIR\MyApp.exeBoth of these lines do exactly the same thing. But the second one is faster because it doesn't check the registry to see what it has to do with the file.

Also, notice how I didn't surround with quotes in the second one. Quotes are only needed if the parameter ($INSTDIR\MyApp.exe) has a space. So Exec "$INSTDIR\My App.exe"would need quotes.
