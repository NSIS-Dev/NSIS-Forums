To know how NSIS scripts look like have a look at the examples. To generate a generic script go to the template generator (http://www.nullsoft.com/free/nsis/makensitemplate.phtml).

Now for your other questions:
1) I don't know how you can tell the version of Apache, but if it is possible NSIS can do it. You can read from the registry, read from an INI file, and read from files. However Apache stores its version, you can read.
2) NSIS does that for you. It won't let the user install unless he has enough free space.
3) Just read the Apache configuration file using FileOpen, FileRead and FileClose. Have a look at this (http://www.clantpa.co.uk/nsis/wiki/index.php/FileRead) example.
4) Use the File command to extract files on the user machine (they will be compressed on your machine). If you want to extract only from a zip file use this (http://www.clantpa.co.uk/nsis/wiki/index.php/ZipDll) extension DLL.
