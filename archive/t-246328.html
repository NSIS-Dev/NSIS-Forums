Hi,

I recently had some problems with a web installer using the SourceForge download mirrors. I started out by including a manual list of mirrors within the installer but found that downloaded files sometimes contained an html error page instead of the expected archive. I've written a plugin which will check a file for the html header found in the error page and parses a list of mirrors from that page.

The plugin can be found at http://nsis.sourceforge.net/SFhelper_Plugin in case anyone else might find this useful.
