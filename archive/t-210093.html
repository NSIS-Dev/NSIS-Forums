$CMDLINE is the command line that was used to execute the installer, not the path to cmd.exe. To get the path to cmd.exe, use:ReadEnvStr $0 COMSPECBut that's not even needed as net is a program and not an internal command of cmd.exe. You can use $SYSDIR\net.exe.
