Define a leave callback function for the custom page and send ${BM_GETSTATE} to the check box to get its state.Function leave
SendMessage $CHECKBOX ${BM_GETSTATE} 0 0 $0
${If} $0 != 0
MessageBox MB_OK checked!
${EndIf}
FunctionEnd
