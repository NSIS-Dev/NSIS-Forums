Put the function itself anywhere in the script and then put this code where you want to check for .NET:

Call IsDotNETInstalled
Pop $0
StrCmp $0 1 +3
MessageBox MB_OK "no .NET bla bla"
Abort

You'd probably want to put this in .onInit.
