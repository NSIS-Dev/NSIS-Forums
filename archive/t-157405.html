In 2.0b4 you can use two directory pages and give each one a different variable to handle. One can handle $INSTDIR and the other $SECONDINSTDIR.

Gaim's installer script (http://cvs.sourceforge.net/viewcvs.py/gaim/gaim/gaim-installer.nsi?rev=1.54&view=auto) is a good example for this. If you're not using the MUI then simply use DirVar and PageEx to get the same outcome.
