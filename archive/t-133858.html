i've been using this

Call ConnectInternet
NSISdl::download /TIMEOUT=30000 http://my.domain.net/my.zip "$TEMP\my.zip"
ZipDLL::extractall "$TEMP\my.zip" "$INSTDIR"
Delete "$TEMP\my.zip"

downloads and unzips the file. using different compression than zip will save you some bytes for the installer aswell, since ther ZipDLL is quite big. you have also the possibility to use UnTGZ (http://nsis.sourceforge.net/archive/viewpage.php?pageid=84) or ExtractDLL (http://nsis.sourceforge.net/archive/viewpage.php?pageid=77)
