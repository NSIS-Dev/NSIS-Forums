When I perform this call with an empty entry it removes the INI entry from the file.

my.ini:
[section]
my_item=""
another_item="abc"

WriteINIStr $INSTDIR\my.ini section my_item ""

Now my.ini looks like this:

[section]
another_item="abc"

This is a very unwanted behavior. Is this a bug?
