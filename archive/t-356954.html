jpderuiter
7th March 2013, 19:43
When using the example (${registry::Find} "$0" $1 $2 $3 $4), $1 will hold the registry path of your program.
This path contains your GUID.
You should be able to extract the GUID using WordFind (http://nsis.sourceforge.net/Docs/AppendixE.html#E.3.2).
