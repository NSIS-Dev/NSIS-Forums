Simply create a file in the root of the cd named autorun.inf. A Sample is shown below.

[autorun]
OPEN=installer.EXE
ICON=\installer.ico
atsuk
21st October 2004, 07:38
explore some original CD's and look for AUTORUN.INF file on the CD. it should look like this:

[autorun]
open=AUTORUN.EXE
icon=ICON.ICO
name=Some name
name2=CdName
