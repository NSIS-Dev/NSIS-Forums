I got the answer.

Use this to disable a custom page ...
GetDlgItem $R0 $HWNDPARENT 3
EnableWindow $R0 0





TO Disable back in finish page

!define MUI_PAGE_CUSTOMFUNCTION_PRE DisableBackButton

!insertmacro MUI_PAGE_FINISH


Function DisableBackButton
GetDlgItem $R0 $HWNDPARENT 3
EnableWindow $R0 0
FunctionEnd
