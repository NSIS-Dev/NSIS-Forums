I'm not sure if anyone remembers but about 2 years ago I made a program in NSIS called D-Day Updater which updated maps (levels) for a Quake 2 mod called D-Day: Normandy.
I then started playing a Half-Life mod called Hostile Intent and ended up making MapUpdater for that (new one from scratch). MapUpdater has since been used in quite a few Half-Life modifications for community map database websites.

I recently had a request to convert MapUpdater into something that will update not just games, but anything. So, here is Software Updata.

I'm probably advertising it a little here, but thats really because the entire program is written in NSIS (minus the NSIS plugins used). All credits are in the manual of course :)

http://swupdata.sourceforge.net

Just something also to show exactly what NSIS is capable of.
Next I'm going to try the skinning plugins on it :)
