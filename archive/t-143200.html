The Windows texts don't depend on the locale you set, they depend on the UI language you install. Just like the start menu doesn't get translated when you change the locale so do the message boxes stay in the same language.

As for a prompt about a missing language, the user should know if his system can or can't display the specific language. Some languages will only miss some characters and the user might know how to read it without those, we can't force him not to read it this way.
