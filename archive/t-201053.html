Code below:

;FUNCTION - ONINIT

Function un.onInit

;Check if the program exe is running before proceeding uninstallation
Call KillProgramExe

FunctionEnd

;**************************************************************************************************
;FUNCTION - KillProgramExe

Function KillProgramExe

FindProcDLL::FindProc "${MUI_PROGRAM_FILENAME}.exe"

foundexe:

notfoundexe:
;MessageBox MB_OK "Program is not running"

FunctionEnd
