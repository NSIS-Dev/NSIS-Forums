Probably the best thing would be the MSSQL OLEDB (http://nsis.sourceforge.net/MSSQL_OLEDB_plug-in) plugin.

You could also use MS's sqlcmd.exe (http://msdn.microsoft.com/en-us/library/ms162773.aspx) and run it using nsExec. (If you are using SQL 2000, then there's also osql.exe (http://technet.microsoft.com/en-us/library/aa213090.aspx) command, although it can be slower than sqlcmd.exe)
