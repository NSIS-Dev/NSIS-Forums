To Check if drive d:\ exists use this command:

IfFileExists D:\*.* DriveOK DriveNotOK

DriveOK
#Code if drive exists
goto DriveCheckEnd

DriveNotOK:
#Code if drive not exists
goto DriveCheckEnd

DriveCheckEnd:

This should fix your problem.
