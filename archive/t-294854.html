Use the "StrFunc" header file. It provides many string manipulation macros!

!include "StrFunc.nsh"
${StrRep}

Section
${StrRep} $0 "$INSTDIR" "\" "/"
MessageBox MB_OK 'Result: "$0"'
SectionEnd

See "Docs\StrFunc\StrFunc.txt" for more info ;)
