Problem is solved as follows:

; [Set focus on Install button]
; Get Item of Install button
GetDlgItem ${TEMP} $HWNDPARENT 1
; Enable Install button
EnableWindow ${TEMP} 1
; Set focus on Install button
SendMessage $HWNDPARENT ${WM_NEXTDLGCTL} ${TEMP} 1
