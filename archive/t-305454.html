CreateDialogFromINI is an nsDialogs pluigin, which is different from the InstallOptions plugin. (For proper use of InstallOptions, refer to the InstallOptions readme file.)

The FAQ section of nsDialogs says this:nsDialogs.nsh contains a function called CreateDialogFromINI that can create nsDialogs' dialog from an INI file. It can handle every type of control InstallOptions supports, but doesn't handle the flags or notification yet. Examples\nsDialogs\InstallOptions.nsi shows a usage example of this function.
