You'll have to create a custom page with InstallOptions with a list box and fill it with GetDrives (in documentation).

You can use this to set the new $INSTDIR:
StrCpy $R0 $INSTDIR
StrCpy $R0 $R0 "" 1
StrCpy $INSTDIR `$DRIVE$R0`
