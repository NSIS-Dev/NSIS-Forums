Hi all,

As a bonus to the helpful NSIS community here NSIS install & un-install Windows Vista compliant icons.

The link below points to a zip file containing Windows Vista look install and un-install icons up to the maximum format of 256x256 pixels. There are two version: one in regular non-compressed icon format and one in the newly compressed icon format of Windows Vista. compressed files have the *.c.ico filename. Use a modern iconeditor to change the design.

Go to: Main location (http://www.mysharefile.com/v/6594930/NSIS_Install_Uninstall_Vista_compliant_icons.zip.html) or directly to the ZIP file (http://www.mysharefile.com/d/6594930/1151591570/NSIS_Install_Uninstall_Vista_compliant_icons.zip)

ALL icons will display in W98,XP,VISTA but the compressed icons will only show the 256x256 size in Windows Vista. The non compressed version will show 256x256 in ALL windows OS's *as long as the OS supports* these large icons.

If it is still confusing to you, read a very good article about large icons
here (http://www.axialis.com/tutorials/tutorial-vistaicons.html)
and
here (http://www.rw-designer.com/vista-icon)

Have fun in creating installers via NSIS with the Windows Vista look.
