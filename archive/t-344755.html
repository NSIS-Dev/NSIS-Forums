We might be able to set these flags in the future, in the mean time you can hex edit the 2.46 stubs. You can find the PE DllCharacteristics flags @ http://msdn.microsoft.com/en-us/library/windows/desktop/ms680339%28v=vs.85%29.aspx You can find the offset with http://wjradburn.com/software/#PEView if you don't have Visual Studio and editbin.
