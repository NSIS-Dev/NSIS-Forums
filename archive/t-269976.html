ExecDos (http://nsis.sourceforge.net/ExecDos_plug-in) with /NOUNLOAD /ASYNC /TOSTACK options puts every output line to NSIS stack instead of log file and does this during application execution. Create loop and update progress bar basing on Pop result. The only problem you should solve for yourself is to find execution end. For progress bar you can use nxs (http://nsis.sourceforge.net/Nxs_plug-in) (which I know is stable) or any other plug-in.
And I am not sure how safe is mutual access to NSIS stack...
