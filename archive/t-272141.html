Exec starts executable, not script, shell required to run vbs. And Forum Search is you best friend ;)
http://forums.winamp.com/showthread.php?s=&threadid=182748&highlight=vbs
If this opens black console window, you can take vbs extension handler from Registry (what ExecShell does) and run it using nsExec (included to distribution) or ExecDos plug-in (from Dev. center (http://nsis.sourceforge.net/Category:Plugins)).
