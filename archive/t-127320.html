If you are using the Finish page (see the WelcomeFinish example), it will automatically be showed when the rebootflag is set.

The reboot flag will automatically be set by the Delete/Rename commands if you are using the /REBOOTOK flag and a reboot is required.

You can also set the reboot flag manually:

SetRebootFlag true
