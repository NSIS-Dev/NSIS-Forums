1. Use ReadRegStr
2. Use GetParent (from NSIS manual)
3. Append "\Skin" by using StrCpy

Function .onInit
ReadRegStr $R0 HKCU "Software\BioDuo" "BioDuo"
Push $R0
Call GetParent
Pop $R0
StrCpy $INSTDIR "$R0\Skin"
FunctionEnd
