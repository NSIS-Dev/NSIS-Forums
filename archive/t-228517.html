Are you sure that doesn't fit in the MUI dialog? The MUI has a bit more space in it than the classic dialog that example uses.

If you'd like to use RECT anyway, open up Contrib\UIs\default.exe in Resource Hacker and take a look at dialog 105. You'll notice a big black rectangle in the middle. That's the default RECT InstallOptions, and any other page uses (id = 1018). The classic dialog (default.exe), doesn't have any other rectangles for InstallOptions to use. Currently, only the MUI dialog (modern.exe), has those for the welcome and finish pages (id = 1044). The id of those rectangles, is the value you should use for the RECT field in the INI.

You can use Resource Hacker to add more rectangles, or even modify the size of the entire dialog. Use MUI_UI to tell the MUI to use your new dialog, or ChangeUI, if you're not using the MUI.
