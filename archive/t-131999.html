If it works on the Windows XP, and only after you transfer it to the Windows 98 machine it stops working then something broke in the transfer process. I am sure that if you move it back to the Windows XP machine it won't work there either. You should probably check your Windows 98 machine for viruses and the connection between the two computers. Just for the comparison, I have a Windows XP machine that I build my installers on and have no problem moving installers from it to my Windows 98 machine or any other machine. No icon is also a good indication of corrupted data, and so is the fact that it opens in a DOS box, which means Windows can't read the EXE headers right.
