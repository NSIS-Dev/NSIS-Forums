Unfortunately, there's not much you can do with a compiled EXE other than run it... (on Windows, that is.)

You can try this forum (http://forums.winamp.com/showthread.php?s=&threadid=207610&highlight=extract) to see if it helps, but I'm not sure it will even work (I've never tested it). And even then, I'm not sure it would even run on Linux.

Another option might be to try to run the installation using WINE (http://www.winehq.com/).

Or, if you have a copy of Windows, you could setup a virtual machine, such as VMWare (http://www.vmware.com/) or QEMU (http://fabrice.bellard.free.fr/qemu/)
