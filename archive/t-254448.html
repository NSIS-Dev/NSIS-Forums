Seems what you want is not a standard INI file...

Look at the Text Function headers (http://nsis.sourceforge.net/Docs/AppendixE.html#E.2) included with NSIS. There are some text file manipulation function that might work better for you. (the LineFind function might be what you are looking for.)

You might also want to browse the NSIS Developer Center (http://nsis.sourceforge.net/Developer_Center) for other plugins/functions that would help you achive your goals.

If you are still stumped, it might help to post a few example lines from your unique INI file to better illustrate what you are trying to accomplish.
