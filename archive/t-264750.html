Start->Programs->NSIS->Contrib->nsExec Readme
By default Vista elevates installers to admin and all child processes have the same level (and this is a problem sometimes). One more DOS plug-in ExecDos (http://nsis.sourceforge.net/ExecDos_plug-in). Forum search (link above) can help you as well.
