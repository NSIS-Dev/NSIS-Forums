Hello there,

I finally got some spare time and created a small plug-in for NSIS. This new plugin is called nsisMultiMon.

nsisMultiMon is a plugin for NSIS (Nullsoft Scriptable Install System) that provides easy multi monitor management for installers. With this plugin, you can determine the number of monitors, get various coordinates and sizes, monitor namings or simply put the complete installer on another connected monitor.

nsisMultiMon was tested on Windows 2000 and Windows XP so far.

I'm using NSIS quite a while and wanted to give something back to the community!

Also I already managed to set up a page in the developer Wiki, but since I'm not a Wiki god, please forgive me some formatting issues, maybe somebody could improve this site a bit :-)

You can find the page here: http://nsis.sourceforge.net/NSISMultiMon_plug-in

A download of the attached file and a link on the plug-ins page on the Wiki page would be also fine ...

Please check this plug-in out and let me know what you think of it.
