Yes, set it's name to an empty string ("") using SectionSetText. For example:

Section "Only on XP" XPSection
# ....
SectionEnd

Function .onInit
# get OS into $0
StrCmp $0 "XP" +3
SectionSetText ${XPSection} ""
SectionSetFlags ${XPSection} 0 # unselect section
FunctionEnd
