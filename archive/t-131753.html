Try something like this:

StrCpy $1 0

loop:
FileOpen $0 "c:\path\to\file" "w"
IfErrors 0 done
IntOp $1 $1 + 1
IntCmpU $1 50 done 0 done # if we got to 50 or passed it, enough
Sleep 500 # wait for 500ms
Goto loop
done:
FileClose $0
