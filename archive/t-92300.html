Just do:File prob$.txtThe "$$" to represent the "$" can only be used within actual strings where variables like $WINDIR and $R0 can be used. These cannot be used in the File argument (since they will only get there value at run time, not at compile time), so it is not necessary to have a $$ option in File. Therefore it doesn't exist :). "$" will be represented by "$".
