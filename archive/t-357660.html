Use the nsJSON plug-in in a small script (ParseJSON.nsi) which will read the json file and output an nsh file (with !defines):OutFile ParseJSON.exe
SilentInstall silent
RequestExecutionLevel user

Section
FileOpen $R0 $EXEDIR\Const.nsh w
nsJSON:: ...
FileWrite $R0 '!define ....'
FileClose $R0
SectionEndThen you can put this in your main script:!system `"${NSISDIR}\makensis.exe" ParseJSON.nsi`
!system `ParseJSON.exe`
!system `del ParseJSON.exe`
!include ParseJSON.nshPersonally I just use an nsh file containing !defines for compile time configuration for basic installers.
