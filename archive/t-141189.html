First of all you will need to use this define line !define MUI_CUSTOMFUNCTION_LICENSE_LEAVE LeaveLicense after that you will need a function similar to thisFunction LeaveLicense
File /oname="$TEMP\readme.txt" "readme.txt"
ExecShell "open" "$TEMP\readme.txt"
FunctionEnd Note that you should not use ExecWait unless you want the user to close down the readme file before installer proceeds and also note that ExecShell will use the default program to open the file. It would also be a good idea to use the ReserveFile command on the readme file to make sure that it is placed at the start of the installer.

Vytautas
