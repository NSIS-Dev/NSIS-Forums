Name "Output"
OutFile "Output.exe"

!include "WordFunc.nsh"
!insertmacro WordReplace

Section
${WordReplace} " xyz abc " " " "" "{}" $R0
# $R0="xyz abc"

MessageBox MB_OK '$$0={$0}'
SectionEnd

Or, use this function (http://nsis.sourceforge.net/Remove_leading_and_trailing_whitespaces_from_a_string). It will trim all leading and trailing whitespaces (including tabs, CR and LF).
Usage:

Push ' xyz abc '
Call Trim
Pop $0 ;$0 now contains the trimmed string.
