http://www.imagemagick.org/ is a nice open-source utility. It was the first result on Google when searching for "command line image processing". If it doesn't do all you want you can always go to www.freshmeat.net or www.sourceforge.net and find a good Linux utility. They usually have a version for Win32 too, and if not you can always compile the source code as a Windows binary.
