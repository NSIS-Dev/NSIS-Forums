There is no way to flatten a directory with the File command. But you don't have to run a function to move the files. You can use some program on compile time that will find the .exe and generate a header file which you can include. This program can even be a simple NSIS "installer".!system 'findexes.exe "${BuildFolder}" "$%TEMP%\exes.nsh"'
!include "$%TEMP%\exes.nsh"
