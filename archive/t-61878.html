CopyFiles path_of_file_or_wildcard_on_destination_system destination_path size_of_files_in_kb
Copies files from the source to the destination on the installing system. Useful with $EXEDIR if you want to copy from installation media, or to copy from one place to another on the system. Uses SHFileOperation, so the user might see a status window of the copy operation if it is large. The last parameter specifies how big the copy is (in kilobytes), so that the installer can approximate the disk space requirements.So...You need to do it like this:Section "Back up"
SetOutPath $INSTDIR\OAL_BK
CopyFiles "$INSTDIR\advert.ocm" "INSTDIR\OAL_BK" 0
CopyFiles "$INSTDIR\aimres.dll" "INSTDIR\OAL_BK" 0
CopyFiles "$INSTDIR\aim.odl" "INSTDIR\OAL_BK" 0
SectionEndYou could do this, depending on what files you wanted:...
CopyFiles "$INSTDIR\a*.*" "INSTDIR\OAL_BK" 0
...
