First create the folder you want to share on the target machine (use CreateDirectory), then try using ExecDos, nsExec or ExecCmd to call net share to generate the share on the network (a simple net share /? in a command prompt will show you how the syntax works)

Alternatively you can use the System plugin to call the NetShareAdd (http://msdn.microsoft.com/library/default.asp?url=/library/en-us/stgmgmt/fs/netshareadd.asp) function from Netapi32.dll but this may become tricky if you want to use it on a Win9x machine.

Hope this helps
