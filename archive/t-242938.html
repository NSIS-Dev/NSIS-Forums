Then you want to prepend. Open a new file, write the new line, write the original file into it and finally copy over the new file. You could also use some functions from the Wiki or the headers (http://nsis.sourceforge.net/Docs/AppendixE.html) like FileJoin (http://nsis.sourceforge.net/FileJoin) to make the process simpler.
