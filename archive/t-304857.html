what should be DBVERSION ?See chapter 3.1 (MakeNSIS Usage) of the NSIS User Manual. (You don't need the DBRAND and DVERSION options)Where do I specify my windows application file name i.e MyApp.exe?You should use the File function to include your application file in the installer.
See chapter 4.9.1.5 of the NSIS User Manual.
There are many examples supplied with NSIS explaining the usage of the File function.
Begin with example1.nsi as a start.
