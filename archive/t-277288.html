MUI_LANGUAGE doesn't set the language, it loads the language files. To set the language, you indeed need to modify $LANGUAGE in .onInit. You can find more information under Language Selection (http://nsis.sourceforge.net/Docs/Chapter4.html#4.10.1) in the documentation.

You can't use language strings in .onInit, the language is initialised after .onInit.

You must use a LogicLib Switch/Select with $LANGUAGE.

${Switch} $LANGUAGE
${Case} ${LANG_ENGLISH}
...
${Break}
${Case} ${LANG_FRENCH}
...
${Break}
${EndSwitch}
