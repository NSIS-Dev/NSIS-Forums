ok, i find it with this script :D

Function "WelcomeColor"
FindWindow $HWND "#32770" "" $HWNDPARENT
StrCpy $0 0
start:
IntOp $0 $0 + 1
GetDlgItem $DLGITEM $HWND $0
StrCmp $DLGITEM "0" start 0
MessageBox MB_OK|MB_ICONSTOP "$0"
SetCtlColors $DLGITEM FFFFFF 4C5844
FunctionEnd
