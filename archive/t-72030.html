Do this on executables and shortcuts

Right Click > Select "Properties" > Click on the "Compatibility" tab and then click the box that says: "Run This Program in Compatibility Mode" then choose which Operating System you want it to be ran on. I did this for an Adobe Acrobat writer installer that would only run on Windows NT4
