If you really want the destination folder fixed (user cannot change it), first callSetOutPath "C:/ProgramData/Autodesk/REVIT/Autodesk2013/Addins"
Then call the File command:File "C:/Users/myName/Desktop/myApplication-2013/*.*"If you want to have a conditional destination directory, you can use LogicLib and SetOutPath: http://nsis.sourceforge.net/LogicLib

However, looking at Revit addin deployment, copying the dll files seems to be not enough, you should also register the addin.
Search google, it seems like some Revit Addin deployment tools are available which makes it easy to create an installer.
