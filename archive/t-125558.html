You could try using FindWindow $0 "Your Window Title"
SendMessage $0 ${WM_KEYDOWN} 13 0(note you have to include WinMessages.nsh to define the WM_KEYDOWN variable). This should send a return key press to the window (13 is the ASCII value of carriage return). (this (http://nsis.sourceforge.net/archive/viewpage.php?pageid=86) might help give you an idea of how SendMessage can be used)

Alternatively something like AutoIt (http://www.hiddensoft.com/AutoIt/) can be used to control applications using key presses and mouse actions.
