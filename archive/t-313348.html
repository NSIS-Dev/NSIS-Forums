ExecShell open "notepad.exe"If you want to run notepad.exe then ExecShell is not the correct command; Exec or ExecWait can be used to start notepad.exe

If you read the manual you will find some examples of how to use ExecShell (http://nsis.sourceforge.net/Docs/Chapter4.html#4.9.1.3). In these examples the action is always enclosed in quotes, e.g.ExecShell "open" "http://nsis.sf.net/"
ExecShell "open" "$INSTDIR\readme.txt"
The first example will use the default browser to display a webpage. The second will display a text file using the default application for this type of file.
