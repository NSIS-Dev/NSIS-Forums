I assume you're using ExecWait to run the AIR installer. And I assume that after that you're using Exec, or something similar, to launch the installed software.

If that is the case, then you probably need to execwait the AIR installer including its child processes. You could try using this: http://nsis.sourceforge.net/ExecWait_and_waiting_for_child_processes_with_Job_Objects
