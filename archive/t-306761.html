if you are using the old InstallOptions(2/Ex) method, just add the "GROUP" value to the FLAGS directive of the -first- radiobutton of each group. e.g.


[Field 2]
Type=RadioButton
Text=Some text
Flags=GROUP|NOTIFY

[Field 3]
Type=RadioButton
Text=Some text
Flags=NOTIFY

[Field 4]
Type=RadioButton
Text=Some text
Flags=GROUP|NOTIFY

[Field 5]
Type=RadioButton
Text=Some text
Flags=NOTIFY
