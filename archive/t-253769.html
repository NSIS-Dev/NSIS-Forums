Yes it is possible,

we can do like this:

;Get the current ShellVarContext
StrCpy $R0 "$SMPROGRAMS"
SetShellVarContext current
StrCmp $R0 "$SMPROGRAMS" 0 +3
StrCpy $R0 "current"
Goto +3
StrCpy $R0 "all"
SetShellVarContext all


$R0 contain "current" or "all"

Enjoy ;)
