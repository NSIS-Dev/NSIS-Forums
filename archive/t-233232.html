If none of the methods described in the FAQ item (http://nsis.sourceforge.net/How_do_I_start/stop/create/remove/check_a_service), you'd have to write a plug-in or call some API using the System plug-in. It seems the ChangeServiceConfig (http://msdn.microsoft.com/library/en-us/dllproc/base/changeserviceconfig.asp) API function is what you need. Maybe it'd be simpler to recreate the service with the preferred start method, after the 3rd party setup ends.
