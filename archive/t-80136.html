Hey there. :)

I believe you're missing the path info. for the file you're trying to rename... Instead of writing:

Rename InvStats.sav InvStats.xml

try:

Rename "$INSTDIR\InvStats.sav" "$INSTDIR\InvStats.xml"

Hope that helps. :up:
