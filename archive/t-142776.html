You should use the NSISdl plug-in. You can find an example in NSIS Update's source code (only available in CVS. You should update to the latest CVS version using NSIS Update or the nightly snapshot or get the source code here (http://cvs.sourceforge.net/cgi-bin/viewcvs.cgi/nsis/NSIS/Contrib/NSIS%20Update/)) and in waplugin.nsi. There are probably some examples in the Archive (http://nsis.sf.net/archive) too.
