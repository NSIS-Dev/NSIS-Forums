If you are that worried about registry entries you can use RegShot (http://regshot.blog.googlepages.com/regshot.html) to get an image of the registry once you start your program (save it somewhere), then use UndoReg (http://neutronstar.cjb.net/) to undo any registry changes. You may have to use something like AutoIt (http://www.autoitscript.com/) to script the whole process since these tools may not offer an unattended mode. However there will always be volatile cache activity that you won't be able to remove ...
