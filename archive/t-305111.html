You can use this script to check if Excel is installed: ClearErrors
EnumRegKey $0 HKCR "Excel.Application" 0
IfErrors error
MessageBox MB_OK "Excel is installed"
Goto done
error:
MessageBox MB_OK "Excel is not installed"
done:
ClearErrorsIt checks if the key "HKEY_CLASSES_ROOT\Excel.Application" exists.
(Idea from http://www.devx.com/vb2themax/Tip/19507)
